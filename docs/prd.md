# 需求文档

## 1. 应用概述

- **应用名称**：258班 班级主页
- **应用描述**：一个面向星火班全体师生的班级主页，纯前端单页 HTML 实现，展示班级核心信息、成员数据与导航入口，内置通讯录、班级成员、时光相册、关于网站四个独立页面，支持亮暗色切换，状态持久化存储于本地浏览器数据库（localStorage）。

---

## 2. 页面结构与功能说明

### 2.1 整体页面结构（树状）

```
班级主页（单页应用，路由切换）
├── 顶部导航栏
├── 主页
│   ├── Hero 区域（校徽 + 班徽 + 学校/班级名称 + 口号 + 班号）
│   ├── 数据卡片区
│   ├── 功能导航区
│   └── 点赞区
├── 通讯录页面
├── 班级成员页面
├── 时光相册页面
├── 关于网站页面
└── 页脚
```

---

### 2.2 顶部导航栏

**初始状态（未滚动）**

- 背景完全透明，无阴影
- 左上角显示班级名称「星火班」，字体醒目
- 右上角显示汉堡菜单展开按钮（三横线图标）

**滚动后状态（页面向下滚动超过 80px）**

- 导航栏变为悬浮固定（position: sticky / fixed），圆角白色背景（亮色模式）/ 圆角深色背景（暗色模式），带轻微阴影
- 圆角值加大，视觉上更为圆润（建议 border-radius: 20px 及以上）
- 过渡动画平滑（背景色渐变过渡）

**汉堡菜单展开后**

- 展开下拉菜单，包含以下导航项：通讯录、时光相册、班级成员、QQ群
- 通讯录、时光相册、班级成员点击后跳转至对应内置页面
- QQ群为占位链接（`#`），点击不跳转
- 点击菜单外区域或再次点击按钮收起菜单

### 2.5 功能导航区

四个导航入口，以图标卡片或大按钮形式展示：

- 通讯录：点击跳转至内置通讯录页面
- 时光相册：点击跳转至内置时光相册页面
- 班级成员：点击跳转至内置班级成员页面
- QQ群：占位链接（`#`），点击不跳转

每个入口包含对应图标与文字标签，视觉风格统一。

### 2.6 点赞区

- 居中展示一个点赞按钮（爱心图标）
- 点击后点赞数 +1，并有动画反馈（顶部爱心飘落，点一下赞飘落一个）
- 显示当前累计点赞数
- 点赞数持久化存储至 localStorage，刷新页面后保留
- 每个用户（同一浏览器）可多次点赞（不限次数）

## 3. 亮暗色切换

- 页面右下角和导航栏内提供亮色/暗色切换按钮
- 切换时全页面平滑过渡
- 当前主题模式持久化存储至 localStorage，下次访问自动恢复
- 默认跟随系统主题（prefers-color-scheme），若本地有存储记录则优先使用存储值
- **暗色模式全局优化要求**：所有页面（主页、通讯录、班级成员、时光相册、关于网站）在暗色模式下均需保证文字可读性、背景层次感与组件配色协调，避免出现白底突兀、对比度不足等问题

## 4. 本地数据持久化

| ------------ | ------------ |

| 当前口号索引 | 0 / 1 / 2    |
| 点赞数       | 累计点赞次数 |

## 5. 业务规则与逻辑

1. **相识天数计算**：以 2023年8月23日 为起始日，计算至当前日期的自然天数（含起始日）。
2. **已毕业天数 / 距毕业天数**：
   - 当前日期 ≥ 2026年7月1日：显示「已毕业 X 天」
   - 当前日期 < 2026年7月1日：显示「距毕业还有 X 天」
3. **口号切换**：循环切换，到达最后一条后回到第一条。
4. **导航链接**：QQ群链接当前为占位（`#`），时光相册四条路线链接当前为占位（`#`），后续可替换为实际地址。
5. **滚动阈值**：页面滚动超过 80px 时触发导航栏背景变化。
6. **复制交互**：点击通讯录表格中的 QQ号、手机号、微信号文字，调用 Clipboard API 复制内容，复制成功后显示轻提示（Toast），持续约 1.5 秒后消失。
7. **QQ头像加载**：通讯录中 QQ 头像通过 `https://q.qlogo.cn/headimg_dl?dst_uin={QQ号}&spec=100` 接口自动加载，加载失败时显示默认占位头像。
8. **页面路由**：各内置页面（通讯录、班级成员、时光相册、关于网站）通过前端路由或显示/隐藏切换实现，无需页面跳转刷新。

## 6. 异常与边界情况

| 场景                         | 处理方式                                                           |
| ---------------------------- | ------------------------------------------------------------------ |
| 本地存储不可用（隐私模式等） | 降级处理，页面正常展示，状态不持久化，不报错                       |
| 图片加载失败                 | 显示占位图或默认图标，不影响页面布局                               |
| 当前日期早于入学日期         | 相识天数显示 0 天                                                  |
| 导航链接点击（占位状态）     | 链接为 `#`，不跳转，加 `event.preventDefault()` 防止页面跳动      |
| 剪贴板 API 不可用            | 复制操作静默失败，不报错，可降级提示用户手动复制                   |
| QQ头像接口加载失败           | 显示默认占位头像，不影响表格布局                                   |
